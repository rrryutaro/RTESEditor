# RTESEditor

TES（The Elder Scrolls）シリーズの ESP/ESM ファイルを編集するための統合エディタです。  
現在は **Morrowind（TES3）** に対応しており、OpenMW の UTF-8 MOD にも利用できます。

---

## ダウンロード（EXE版）

Python のインストール不要で使えるスタンドアロン EXE は、[Releases](../../releases) からダウンロードできます。

---

## 特徴

- ESP / ESM バイナリファイルの読み込み・表示・編集・保存
- 複数ファイルの同時読み込みと競合表示（Conflict View）
- エンコーディング対応：CP1252（英語版）/ Shift-JIS（日本語版）/ UTF-8（OpenMW）
  - ファイル読み込み時に自動判定
  - ファイルごとに個別指定可能
- Shift-JIS 日本語版特有のトレイルバイト `0x40 ↔ 0x7F` 変換に対応
- レコードタイプごとの列表示カスタマイズ（列ヘッダ右クリックで設定・永続保存）
- TSV エクスポート / インポート
- テーマ切り替え（標準 / ダーク）
- フォント設定
- 検索（Ctrl+F）
- 常に最前面表示

## 使い方

詳細は [Wiki](../../wiki) を参照してください。

### ファイルの読み込み

「ファイル → 開く」からファイルを追加します。複数ファイルを同時に登録できます。

| 列 | 説明 |
|----|------|
| エンコード | ファイルのエンコーディング（自動判定後に変更可） |
| 上書き | 複数ファイルで同一レコードが競合した際にこのファイルを優先表示する |
| 保存対象 | 「保存」「名前を付けて保存」の保存先とし、競合ビューでの編集を許可する |
| 検索対象 | 全体検索の対象とする |

### 保存

| メニュー | 動作 |
|---------|------|
| ファイル → 保存 | 「保存対象」ファイルを元のパスに上書き保存（確認ダイアログあり） |
| ファイル → 名前を付けて保存 | 「保存対象」ファイルを別名で保存 |
| ファイル → 修正箇所のみ書き出し | 今セッションで編集したレコードのみを指定ファイルへ書き出す。既存ファイルを指定した場合はマージ（既存 + 新修正） |

### TSV エクスポート / インポート

「ファイル → エクスポート (TSV)」「ファイル → インポート (TSV)」で一括編集が可能です。

## 動作環境（Python 版）

- Python 3.11 以降
- PySide6

## インストール（Python 版）

```bash
pip install PySide6
```

## 起動方法（Python 版）

```bash
python main.py
```

---

## ビルド（EXE 生成）

```bash
pip install pyinstaller
cd RTESEditor
pyinstaller RTESEditor.spec --noconfirm
```

または `RTESEditor/build_exe.bat` をダブルクリックしてください。  
生成された EXE は `RTESEditor/dist/RTESEditor.exe` に出力されます。
